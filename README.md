# Fake News Detection using NLP
NLP-based machine learning project to classify news articles as fake or real using TF-IDF and Logistic Regression.


## Overview
This project classifies news articles as fake or real using Natural Language Processing techniques.

## Tools Used
- Python
- Scikit-learn
- TF-IDF Vectorizer

## Models
- Logistic Regression

## Outcome
The model effectively identifies fake news articles.
